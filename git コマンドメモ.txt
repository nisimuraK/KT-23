git init            :「git init」コマンドはGitリポジトリを新たに作成するコマンド。

★git add ファイル名  :ファイルの作成/変更/削除をgitのインデックスに追加

★git commit -m "コメント" :変更結果をローカルリポジトリにコミットする（git commit）

★git push origin master :ローカルリポジトリをプッシュしてリモートリポジトリへ反映させる

git status :リポジトリの状態を確認するために使うコマンド
現在のブランチの名称や、追加・変更されたファイルやディレクトリの一覧を表示。

git remote add origin URL :リモートリポジトリの情報を追加。(どのリポジトリにアップロードするか)

git log :ローカルリポジトリのコミット履歴を閲覧するために使うコマンド

git clone :GitHubに公開されているリポジトリを自分のコンピュータへ落とすときに使う。

git reset:ローカルリポジトリのコミットを取り消すために使うコマンド

git remoteb :リモートリポジトリを操作するために使うコマンドで下記のように使う。
git remote：リモートリポジトリの名称一覧を表示
git remote -v：リモートリポジトリの詳細一覧を表示
git remote add [name] [url]：リモートリポジトリを追加
git remote rm [name]：リモートリポジトリを削除

★…特によく使う
